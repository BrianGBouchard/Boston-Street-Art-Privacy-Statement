"Boston Street Art" requests the user's location in order to help the user more easily locate street art installations throughout the city in relation to his or her location. This is also helpful for creating a new entry at the user's current location. The user can enable and disable location services in settings. "Boston Street Art" requests access to the user's photo library and/or camera so that the user can select a photo for a given entry. The user can select a photo or take a new photo to add to the map.
